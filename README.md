This repo. notifies Unilever's brand-team as soon as a new packshot/hero image appears on its USHOPBD Shopify portal, in order to ensure quality online presence.<br>
Tech stack: ```Google Cloud Storage``` ```MD5 Hashing``` ```BeautifulSoup/bs4``` ```Cron job``` ```Linux - Ubuntu``` ```GitHub Actions```<br>
